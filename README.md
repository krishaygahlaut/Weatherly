# Weatherly ☁️🌤

A beautiful iOS SwiftUI Weather App built by Krishay Gahlaut with:

- 🔍 Search by City
- 📍 Real-time GPS Weather
- 🌦 Full-screen Animated Effects (Rain, Sun, Wind, Clouds)
- 🌙 Light/Dark Mode Toggle
- 🚀 Built in SwiftUI + CoreLocation
